Linux-查看进程和删除进程

发表于 2017-12-03 更新于 2021-02-03
  1. 在 LINUX 命令平台输入 1-2 个字符后按 Tab 键会自动补全后面的部分(前提是要有这个东西，例如在装了 tomcat 的前提下, 输入 tomcat 的 to 按 tab)。
  2. ps 命令用于查看当前正在运行的进程。

    grep 是搜索

    例如： ps -ef | grep java

    表示查看所有进程里 CMD 是 java 的进程信息

    ps -aux | grep java

-aux 显示所有状态

ps
3. kill 命令用于终止进程

例如： kill -9 [PID]

-9 表示强迫进程立即停止

通常用 ps 查看进程 PID ，用 kill 命令终止进程
