Centos中安装PHP的PDO-MySQL扩展

December 3, 2017 · 1 min · ZhaoGuibin

1、下载 文件 或者 进入 在PHP源码包中进入ext/pdo_mysql http://pecl.php.net/get/PDO_MYSQL-1.0.2.tgz

2、解压文件

tar zxvf PDO_MYSQL-1.0.2.tgz

3、配置和编译文件

cd PDO_MYSQL-1.0.2
/usr/local/php/bin/phpize
./configure –with-php-config=/usr/local/php/bin/php-config –with-pdo-mysql=/usr/local/mysql
make
make install

注： 我的PHP安装在 ： /usr/local/php/ mysql 安装在 ： /usr/local/mysql 编译的时候注意你自己的安装目录在哪里 3、安装到PHP配置下

把这个记住，然后打开 php.ini文件， 并添加一行

extension=pdo_mysql.so
